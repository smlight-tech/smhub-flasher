One-time USB driver setup

Windows needs permission to access the SMHUB over USB. Click Install — you'll see a standard Windows UAC prompt. Once approved, the driver will install silently in the background. 

cat << 'EOF' | sudo tee /etc/udev/rules.d/99-smhub-flasher.rules
SUBSYSTEM=="usb", ATTR{idVendor}=="3346", ATTR{idProduct}=="1000", MODE="0666"
SUBSYSTEM=="usb", ATTR{idVendor}=="18d1", ATTR{idProduct}=="4ee0", MODE="0666"
EOF
sudo udevadm control --reload-rules
sudo udevadm trigger
